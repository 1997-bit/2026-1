**Salario base quincenal:** B/.260.00

**Horas Extras — Semana 1** (lunes 4h, jueves 2h → 3 diurnas + 1 nocturna + 2 diurnas)

> El taller asigna un día feriado en la semana; las horas tras la jornada normal se clasifican como diurnas (1.25x) o nocturnas (1.50x).

$$\text{Hora extra diurna: } 1.25 \times 3.00 = 3.75$$

$$\text{Hora extra nocturna: } 1.50 \times 3.00 = 4.50$$

$$\text{Semana 1: } (3.75 \times 3) + (4.50 \times 1) = 11.25 + 4.50 = B/.15.75$$

**Horas Extras — Semana 2** (martes 2h, miércoles 3h, viernes 2h = 7h diurnas)

$$3.75 \times 7 = B/.26.25$$

$$\text{Total Horas Extras} = 15.75 + 26.25 = B/.42.00$$

### Empleado 3 — Marcopolo | Gerente de Sucursal

**Salario base mensual:** B/.3,000.00 → quincenal: B/.1,500.00

**Dieta mensual:** B/.700.00 → quincenal: B/.350.00

> La dieta de B/.700.00 **no supera** el 25% del salario mensual (25% × 3,000 = B/.750.00), por lo tanto **no es base para CSS ni ISR** (Art. 91 CSS). Se paga aparte.
